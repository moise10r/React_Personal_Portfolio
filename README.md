## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone git@github.com:moise10r/React_Personal_Portfolio.git
cd ./React_Personal_Portfolio
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### Build

```bash
npm run build
```
